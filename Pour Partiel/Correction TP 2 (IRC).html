Exercice 1

Pour chacun des extraits de code suivants, exprimez à l’aide d’un O(f(n))\mathcal{O}(f(n))O(f(n)) le nombre d’opérations élémentaires effectuées (on négligera le temps passé par l’instruction for elle-même) :

  1. Programme 1

   test = test + 1

   test = test - 1

    💡 En Python, range(n) renvoie l’intervalle des nombres entiers [0;n−1][0 ; n - 1][0;n−1].

    Solution :
    T(n)=1+2n+2n=O(n)T(n) = 1 + 2n + 2n = \mathcal{O(n)}T(n)=1+2n+2n=O(n).

  2. Programme 2

  • pour Z\mathbb{Z}Z, il suffit (mais il existe d’autres possibilités !) de numéroter les entiers positifs et négatifs en alternant : 0→0,1→1,−1→20→0, 1→1, −1→20→0,1→1,−1→2, etc., ou, de manière plus explicite :
    {∣2n∣ si n≤02n−1 si n≥0 \begin{equation} \begin{cases} |2n| \text{ si } n \leq 0\\ 2n-1 \text{ si } n \geq 0 \end{cases} \end{equation} {∣2n∣ si n≤02n−1 si n≥0​​​
    Cette fonction est bien injective, puisque deux entiers relatifs distincts auront pour images deux entiers naturels distincts ; elle est également surjective puisqu’à tout entier naturel on peut faire correspondre au moins un entier relatif ;

  • pour Q\mathbb{Q}Q, c’est un peu plus complexe, car on a “deux degrés de libertés” (le numérateur et le dénominateur) ; on ne peut pas simplement lister tous les rationnels avec un numérateur égal à 1, puis ceux avec un numérateur égal à 2, etc. car il en existe à chaque fois une infinité. Donc si on commençait par lister tous les rationnels avec un numérateur égal à 1, on n’atteindrait jamais les autres.
    A la place, pour les rationnels positifs, Cantor a trouvé cette méthode astucieuse, consistant à parcourir les rationnels en “zig-zaguant” (on saute les fractions réductibles, car on a déjà rencontré ces nombres auparavant)

    On peut facilement prouver qu’il s’agit d’une bijection.
    Enfin, pour prouver que tous les rationnels (positifs et négatifs) sont dénombrables, il suffit d’utiliser la même stratégie que celle utilisée pour Z\mathbb{Z}Z : on réordonne les rationnels en commençant par 000, puis en alternant les positifs et négatifs, ce qui donne : {01,11,−11,12,−12,21,−21,31,−31…}\{{\frac{0}{1},\frac{1}{1},\frac{-1}{1},\frac{1}{2},\frac{-1}{2},\frac{2}{1},\frac{-2}{1},\frac{3}{1},\frac{-3}{1}…}\}{10​,11​,1−1​,21​,2−1​,12​,1−2​,13​,1−3​…}
